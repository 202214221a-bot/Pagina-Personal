Temas Aprendidos en Programación Web

HTML5

HTML5 es el lenguaje de marcado utilizado para crear la estructura de una página web. Permite organizar contenido mediante etiquetas como encabezados, párrafos, imágenes, tablas, formularios, enlaces y contenido multimedia.

CSS3

CSS3 se utiliza para dar diseño y estilo a las páginas web. Permite modificar colores, tamaños, bordes, fondos, alineaciones, animaciones y hacer que una página se vea más profesional y atractiva visualmente.

Enlaces Relativos y Absolutos

Los enlaces relativos permiten conectar páginas internas dentro del mismo proyecto web, mientras que los enlaces absolutos permiten acceder a páginas externas como GitHub, W3Schools o YouTube utilizando URLs completas.

Formularios HTML

Los formularios permiten recopilar información del usuario mediante campos como nombre, correo electrónico, teléfono, comentarios, listas desplegables, botones y casillas de selección.

Tablas en HTML

Las tablas sirven para organizar información en filas y columnas. Son útiles para mostrar cronologías, horarios, registros y datos académicos de manera clara y ordenada.

Contenido Multimedia

HTML permite insertar imágenes, videos, audios e iframes. Esto ayuda a hacer una página más interactiva y visualmente atractiva para el usuario.

GitHub

GitHub es una plataforma que permite almacenar proyectos, compartir código y trabajar de forma organizada en desarrollo de software. Es muy importante para publicar proyectos web profesionales.

Visual Studio Code

Visual Studio Code es el editor de código que utilizamos para desarrollar páginas web. Permite escribir código HTML, CSS, JavaScript y organizar proyectos de manera eficiente.
